# Grape-Leaf-Disease-Classification

Deep learning models to classify grape leaf diseases into 4 categories: Blackrot, Esca (Black Measles), Healthy, and Leafblight (Isariopsis Leaf Spot).

Features
- Custom CNN Model: From-scratch convolutional neural network achieving ~97% validation accuracy.

- VGG16 Transfer Learning: Pre-trained VGG16 model reaching ~98% accuracy, outperforming custom CNN.


Results
- Training graphs and confusion matrix visualize performance.

- Custom CNN offers fast training; VGG16 provides highest accuracy.
